# Roadmap

- v0.1: terminal foundation, config, provider chat, Skill Lens, built-in tool execution, remote-ready skill registry foundation, npm installer scaffold, Axiom Coder v0.1, and Proof Mode v0.1.
- v0.2: stronger project editing workflows and richer patch application.
- v0.3: richer proof analytics, safer multi-step workflows, and optional proof sharing/export polish.
- v0.4: safe update checks and controlled skill update installation.
- Later: external skill binary model, remote registry publishing flow, and app layers only after the CLI is stable.

The future `axiom-skills` repository is separate from this Axiom Agent repo. The current local registry fixture stays in-tree for tests, examples, and offline fallback.
